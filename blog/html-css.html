Week Two Post: Box Model Learning

1 July 2021


Every element you put on a webpage is a box, even text. The space the text takes up is a box.
You can see this happen when when you add some CSS colours, you can see the square box shape around the text.
The box model helps you define spaces between elements. This gives you control over the layout and style. It can be
applied to images, shapes, and text. The box model defines the different parts of that shape, which are broken
up into four pieces, or sub-boxes:

Content:

The inner content of your HTML element. For example if you typed text into a paragraph element, this is the area where the text is displayed.



Padding:

A box that surrounds an HTML element's inner content. It's the space that extends slightly outside of the content.

Border:

A box that surrounds an HTML element's padding. Going out further away from the inner content. In the image below, it's the darker space around the padding:

Margin:

the invisible box that surrounds the HTML element's border. It's used as a way to create white space to seperate HTML elements from each other.


The box elements can be adjusted, left, right, top, bottom, width & height to suit your layout needs. However the padding will
never go over the border properties.
Examples of the adjustments that can be applied to image, text, or shapes by themselves or together:
padding-left: 200px; padding-right:20px; border-width: 5px.
