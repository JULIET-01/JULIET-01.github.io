Box Model

1 July 2021


Every element you put on a webpage is a box, even text. The space the text takes up is a box.
You can see this happen when when you add some CSS colours, you can see the square box shape.
The box model defines the different parts of that shape, which are broken up into four pieces, or sub-boxes:

Content:

the inner content of your HTML element. For example if you typed text into a element, this is the area where the text is displayed

Padding:

a box that surrounds an HTML element's inner content. In the example you can see the extra space around the text that also has the same colour.
It's the space that extends slightly outside of the content.

Border:

a box that surrounds an HTML element's padding. Going out further away from the inner content. In the image below, it's the darker space around the padding:

Margin:

the invisible box that surrounds the HTML element's border. It's used as a way to create white space to seperate HTML elements from each other.
The example shows the white space between the two elements, this is the margin.
